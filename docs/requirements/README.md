## 概要

### requirements-1.md（メイン要件定義）
- プロジェクト概要
- ステークホルダーとロール
- 機能要件（認証、資機材管理、予約機能、通知、監査ログなど）
- データ要件
- 非機能要件（セキュリティ、パフォーマンス、可用性など）

### requirements-2.md（データ連携）
- Google BigQuery / Cloud SQL / Sheets へのデータ同期
- 増分同期アーキテクチャ
- 実装ステップと運用注意事項

### requirements-3.md（次期開発項目）
- **中優先度**: CSV エクスポート、ユーザー向け予約履歴画面、ダッシュボード改善
- **低優先度**: 通知機能、監査ログ、パスワードポリシー、機材別名フィールド
